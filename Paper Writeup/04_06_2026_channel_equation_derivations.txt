===========================================================================
  DERIVATION AND ASSUMPTIONS BEHIND CHANNEL MODEL EQUATIONS (Eq. 4-8)
===========================================================================

This document traces each equation in the Channel Propagation Model
subsection back to its theoretical origin, the measurement campaigns
or physical principles from which it was derived, and the key
assumptions that must hold for the equation to be valid.


===========================================================================
EQUATION 6 (eq:pl_free) — FREE-SPACE PATH LOSS  [Start here — foundational]
===========================================================================

  PL_free(d) = 20*log10(d) + 46.4 + 20*log10(fc/5)

DERIVATION:
  This is a rearrangement of the Friis free-space transmission equation.
  Starting from Friis (1946):

    P_r / P_t = (lambda / (4*pi*d))^2

  Converting to dB path loss:

    PL = 10*log10(P_t/P_r) = 20*log10(4*pi*d/lambda)

  Substituting lambda = c/f:

    PL = 20*log10(d) + 20*log10(f) + 20*log10(4*pi/c)

  Now, 20*log10(4*pi/c) with c = 3e8 m/s and f in Hz:

    20*log10(4*pi/(3e8)) = 20*log10(4.189e-8) = -147.56 dB

  So:  PL = 20*log10(d) + 20*log10(f) - 147.56   [f in Hz, d in m]

  The ITU-R M.2135 form normalizes to fc in GHz:
    20*log10(f) = 20*log10(fc * 1e9)
               = 20*log10(fc) + 180

  Then further normalizes to 5 GHz reference:
    20*log10(fc) = 20*log10(fc/5) + 20*log10(5)
                 = 20*log10(fc/5) + 13.98

  Combining constants:
    PL = 20*log10(d) + 180 - 147.56 + 13.98 + 20*log10(fc/5)
       = 20*log10(d) + 46.42 + 20*log10(fc/5)

  Rounded: PL = 20*log10(d) + 46.4 + 20*log10(fc/5)   [QED]

ASSUMPTIONS:
  - Isotropic antennas (antenna gains absorbed into link budget elsewhere)
  - Far-field propagation (d >> lambda; at 5.9 GHz, lambda ~5 cm, so d >> 5 cm)
  - No reflections, diffraction, or scattering — line-of-sight only
  - Free space (no ground plane, buildings, or atmosphere)

SOURCE: Friis, H.T. (1946). "A Note on a Simple Transmission Formula."
        Proceedings of the IRE, 34(5), 254-256.
        Standardized form: ITU-R M.2135-1, Annex 1, Table A1-2.

EQUATION 5 (eq:breakpoint) — BREAKPOINT DISTANCE
===========================================================================

  d_BP = 4 * h_TX * h_RX * fc / c

DERIVATION:
  This comes from the two-ray ground reflection model. When a transmitter
  and receiver are above a reflective ground plane, the received signal is
  the vector sum of the direct (LOS) ray and the ground-reflected ray.

  The path length difference between direct and reflected rays is:

    delta_d = sqrt((h_TX + h_RX)^2 + d^2) - sqrt((h_TX - h_RX)^2 + d^2)

  For d >> h_TX, h_RX (far field), using binomial approximation:

    delta_d ≈ 2 * h_TX * h_RX / d

  The phase difference between the two rays is:

    delta_phi = 2*pi * delta_d / lambda = 4*pi * h_TX * h_RX / (lambda * d)

  The breakpoint distance is defined as the distance where this phase
  difference equals pi (first destructive interference null):

    delta_phi = pi  =>  d_BP = 4 * h_TX * h_RX / lambda

  Since lambda = c/fc:

    d_BP = 4 * h_TX * h_RX * fc / c     [QED]

  At d < d_BP:  The direct and reflected rays combine approximately
                constructively, and path loss follows ~d^2 (exponent ~2)
  At d > d_BP:  The rays increasingly cancel, and path loss transitions
                to ~d^4 (exponent ~4)

  This is why the LOS model (Eq. 4) has two slopes: n ≈ 2.2 before the
  breakpoint and n = 4.0 after.

ASSUMPTIONS:
  - Flat, smooth ground plane with reflection coefficient ≈ -1
  - h_TX and h_RX are "effective" antenna heights above the ground
    (in ITU-R M.2135: h'_TX = h_TX - 1.0m, h'_RX = h_RX - 1.0m,
     where 1.0m is the "environment height" accounting for cars/obstacles)
  - The code (computeUrbanMicro) uses: (hNodeB_ - 1) and (hUe_ - 1)
    for both breakpoint and height-dependent terms
  - d >> max(h_TX, h_RX) for the binomial approximation to hold
  - Single ground reflection dominates (no side-wall reflections)

  For V2X at 5.9 GHz with h_TX = h_RX = 1.5m (vehicle antenna height):
    h' = 1.5 - 1.0 = 0.5m
    d_BP = 4 * 0.5 * 0.5 * 5.915e9 / 3e8 = 19.7 m
  This is quite short, meaning the dual-slope transition happens early.

  Code reference: LteRealisticChannelModel.cc:2306-2307

SOURCE: Two-ray ground reflection model (classical electromagnetics).
        Formalized for cellular: Xia, H.H. (1997). "A Simplified
        Analytical Model for Predicting Path Loss in Urban and Suburban
        Environments," IEEE Trans. Veh. Tech., 46(4).
        Standardized: ITU-R M.2135-1, Annex 1, Table A1-2, Note 1.

EQUATION 4 (eq:pl_los) — LOS DUAL-SLOPE PATH LOSS MODEL
===========================================================================

  PL_LOS(d) = { 22*log10(d) + 28 + 20*log10(fc),          d < d_BP
              { 40*log10(d) + 7.8 - 18*log10(h'_TX)
              {   - 18*log10(h'_RX) + 2*log10(fc),        d >= d_BP

  where h'_TX = h_TX - 1.0m and h'_RX = h_RX - 1.0m are effective
  antenna heights (subtracting 1.0m "environment height" for cars/
  street-level obstacles per ITU-R M.2135-1).

  Code reference: LteRealisticChannelModel.cc:2326-2330
  (computeUrbanMicro function, LOS branch)

  NOTE: The previous version of this file incorrectly listed the
  Analytical model coefficients (22.7, 27, 7.56, 17.3, 2.7). The
  simulation uses the URBAN_MICROCELL scenario, not ANALYTICAL. The
  Analytical model (LteRealisticChannelModel.cc:1769-1773) has slightly
  different coefficients but is NOT used in our simulation config.

DERIVATION:
  This is the ITU-R M.2135-1 / 3GPP TR 36.843 Urban Microcell (UMi)
  LOS path loss model, derived from a combination of:

  (a) Physical basis: The two-ray ground reflection model predicts a
      dual-slope behavior with path loss exponents n1 ≈ 2 (before
      breakpoint) and n2 ≈ 4 (after breakpoint).

  (b) Empirical calibration: The WINNER II project (IST-4-027756,
      Deliverable D1.1.2, Feb 2008) conducted extensive measurement
      campaigns across European cities at frequencies from 2-6 GHz.
      The measured data was fitted to a dual-slope log-distance model
      with frequency-dependent intercepts.

  The general form of a log-distance path loss model is:

    PL = 10*n*log10(d) + A + B*log10(fc) + C

  BEFORE BREAKPOINT (d < d_BP):
    - Fitted path loss exponent: n1 = 2.2 (close to free-space n=2)
    - The coefficient 22 = 10 * 2.2
    - Intercept: 28 + 20*log10(fc) calibrated from measurements
    - The 20*log10(fc) term matches free-space frequency dependence
    - Physical meaning: Mostly free-space propagation with minor
      ground reflection effects and waveguide-like street channeling

  AFTER BREAKPOINT (d >= d_BP):
    - Fitted path loss exponent: n2 = 4.0 (matches two-ray theory)
    - The coefficient 40 = 10 * 4.0
    - Height-dependent terms: -18*log10(h'_TX) and -18*log10(h'_RX)
      capture the antenna height gain (higher antennas -> less path
      loss). These use EFFECTIVE heights (h - 1.0m).
    - The 2*log10(fc) is much weaker frequency dependence than
      before the breakpoint, consistent with ground reflection
      dominated propagation
    - 7.8 is the fitted intercept constant

  The model ensures continuity at d = d_BP (both expressions give
  the same PL value at the breakpoint).

  The model is lower-bounded by PL_free (Eq. 6) — if the dual-slope
  formula gives a value less than free-space loss, free-space loss is
  used instead (this prevents unphysical negative "excess" path loss
  at very short distances).

ASSUMPTIONS:
  - Urban microcell deployment: BS and UE both BELOW rooftop
  - Street-canyon propagation (Manhattan-like grid layout)
  - LOS condition: clear optical sight line between TX and RX
  - Valid frequency range: 2-6 GHz (5.9 GHz is at the edge)
  - Valid distance range: 10 m < d < 5000 m (a lower bound avoids
    near-field effects; upper bound is practical UMi range)
  - Antenna heights: BS at 10m, UE at 1.5-2.5m (original calibration)
  - For V2X: Both antennas at ~1.5m (vehicle rooftop), which is
    outside the original calibration range but commonly used in V2X
    literature as the best available standardized model

LINEAGE:
  Measurement campaigns → WINNER II D1.1.2 (2008) → adopted into
  ITU-R M.2135-1 (2009) → adopted into 3GPP TR 36.814 (2010) →
  used in 3GPP TR 36.843 (D2D studies) → used in openCV2X simulator

SOURCE: ITU-R Report M.2135-1 (12/2009), Table A1-2, "Urban micro-cell"
        WINNER II Deliverable D1.1.2, v1.2 (Feb 2008), Table 4-4.

EQUATION 7 (eq:pl_nlos) — NLOS URBAN MICROCELL PATH LOSS
===========================================================================

  PL_NLOS(d) = 36.7*log10(d) + 22.7 + 26*log10(fc)

  *** This is the equation the user is most interested in ***

DERIVATION:
  This is the ITU-R M.2135-1 Urban Microcell NLOS path loss model.
  Unlike the LOS model (which has a strong physical basis in the two-ray
  model), the NLOS model is primarily EMPIRICAL — derived from fitting
  measurement data to a log-distance model.

  STEP 1: MEASUREMENT CAMPAIGNS
    The WINNER II project conducted channel sounding measurements in
    urban microcell NLOS environments across European cities (Helsinki,
    Munich, Madrid, Oulu, among others) at frequencies 2-6 GHz. The
    NLOS measurements were taken at locations where the receiver was
    around a street corner from the transmitter — no direct line of
    sight, with signal reaching the receiver via:
      - Diffraction around building corners
      - Reflection off building facades across the street
      - Scattering from street furniture, vehicles, etc.
      - Transmission through buildings (minor contribution)

  STEP 2: LOG-DISTANCE MODEL FIT
    The measured data was fitted to the standard log-distance form:

      PL = 10*n*log10(d) + A + B*log10(fc)

    where n is the path loss exponent, A is the intercept, and B
    captures frequency dependence. The least-squares fit yielded:

      n = 3.67  →  10*n = 36.7
      A = 22.7
      B = 26

  STEP 3: INTERPRETING THE COEFFICIENTS

    36.7*log10(d):  Path loss exponent n = 3.67
    ─────────────
      - Free space: n = 2.0
      - LOS street canyon: n ≈ 2.27 (waveguide effect reduces loss)
      - NLOS around corner: n = 3.67 (significantly more attenuation)
      - Two-ray beyond breakpoint: n = 4.0

      The value 3.67 is physically reasonable: NLOS propagation in
      urban microcells suffers from additional loss due to diffraction
      and multiple reflections, but it's not as severe as the n=4
      two-ray post-breakpoint case because multipath reflections from
      building facades partially compensate for the loss of the direct
      path. The dominant mechanism is typically single-bounce corner
      diffraction, which adds roughly 10-20 dB of excess loss compared
      to LOS at the same distance.

    22.7 (intercept):
    ─────────────
      This is the same intercept as in the LOS model's first slope
      (22.7*log10(d) + 27 + 20*log10(fc)), which is NOT a coincidence.
      The WINNER II team constrained the NLOS model to be consistent
      with the LOS model at short distances / corner diffraction
      geometry. The intercept effectively sets the "reference loss"
      at d = 1m.

    26*log10(fc):  Frequency dependence
    ─────────────
      - Free space: 20*log10(f) — from wavelength scaling
      - LOS (pre-BP): 20*log10(fc) — same as free space
      - NLOS: 26*log10(fc) — STRONGER frequency dependence

      The coefficient 26 (vs. 20 for free space) means NLOS attenuation
      increases faster with frequency than free-space loss alone. This
      is because:
        (a) Diffraction loss increases with frequency (shorter wavelength
            diffracts less efficiently around building corners)
        (b) Reflection coefficients of building materials are frequency-
            dependent
        (c) Scattering from sub-wavelength objects increases at higher
            frequencies

      The extra 6 dB/decade (26 vs 20) of frequency dependence is a
      hallmark of NLOS urban propagation.

  STEP 4: WHAT THE EQUATION OMITS (simplifications in the simulator)
    The full ITU-R M.2135-1 UMi NLOS formula actually includes a
    height-dependent correction term:

      PL_NLOS = 36.7*log10(d) + 22.7 + 26*log10(fc) - 0.3*(h_m - 1.5)

    where h_m is the mobile station height. The -0.3*(h_m - 1.5) term
    is omitted in the simulator, which implicitly assumes h_m = 1.5m
    (making the correction zero). This is appropriate for V2X where
    vehicle antenna height is ~1.5m.

  STEP 5: SHADOW FADING
    The model specifies a log-normal shadow fading standard deviation
    of sigma = 4 dB for UMi NLOS. This is applied separately in the
    simulator as a random component on top of the deterministic path
    loss equation.

ASSUMPTIONS (CRITICAL):
  1. Manhattan-grid street layout: Streets are roughly perpendicular.
     The NLOS path goes around ONE corner. If the receiver is two or
     more corners away, additional loss beyond this model is expected.

  2. Both antennas BELOW rooftop: Neither TX nor RX can "see over"
     buildings. The signal propagates along street canyons and around
     corners, not over rooftops. This is a key difference from Urban
     Macrocell (UMa) models.

  3. Outdoor-to-outdoor only: Both TX and RX are outdoors. No
     building penetration loss is included.

  4. Single-slope model: Unlike the LOS model, the NLOS model has
     NO breakpoint / dual-slope behavior. The physical justification
     is that NLOS propagation is dominated by diffraction and
     scattering (not ground reflection), so the two-ray breakpoint
     phenomenon doesn't apply.

  5. Valid frequency range: 2-6 GHz. At 5.9 GHz (V2X band), the
     model is being used at its upper calibration boundary.

  6. Valid distance range: 10 m < d < 2000 m (some sources say 5000 m,
     but the measurements were primarily within 2 km).

  7. Flat terrain: No hills or elevation changes.

  8. BS height = 10m, UE height = 1.5m in original calibration.
     For V2X (both at 1.5m), the model is used outside its original
     calibration but is widely accepted in V2X simulation literature
     as the best available standardized NLOS model.

  9. The model does NOT distinguish between:
     - First perpendicular street (one corner away)
     - Second perpendicular street (two corners away)
     - Parallel street behind buildings
     All NLOS geometries are lumped into a single statistical model.

WHY THIS MODEL IS APPROPRIATE FOR V2X INTERSECTION SIMULATION:
  - At an intersection, cross-street vehicles are in NLOS until they
    reach the intersection itself. The dominant propagation mechanism
    is exactly what this model captures: diffraction around building
    corners and reflection off facades.
  - The UMi scenario (below-rooftop) matches V2X antenna heights.
  - The 5.9 GHz V2X band is within (barely) the calibration range.
  - This is the same model used by 3GPP for D2D/sidelink evaluation
    in TR 36.843, so it has institutional precedent for V2X use.

LINEAGE:
  COST 231 Walfish-Ikegami (1991) → WINNER I B1 NLOS (2005) →
  WINNER II D1.1.2 B1 NLOS (2008) → ITU-R M.2135-1 UMi NLOS (2009) →
  3GPP TR 36.814 (2010) → 3GPP TR 36.843 (D2D, 2014) → openCV2X

SOURCE:
  ITU-R Report M.2135-1 (12/2009), Annex 1, Table A1-2.
  WINNER II Deliverable D1.1.2, v1.2 (Feb 2008), Table 4-4, Scenario B1.
  3GPP TR 36.814 V9.2.0 (2017-03), Table A.2.1.1-2.

EQUATION 8 (eq:nakagami) — NAKAGAMI-m FADING PDF
===========================================================================

  f(P) = (m^m * P^(m-1)) / (Gamma(m) * Omega^m) * exp(-m*P/Omega)

DERIVATION:
  The Nakagami-m distribution was introduced by Minoru Nakagami in his
  1960 paper "The m-Distribution — A General Formula of Intensity
  Distribution of Rapid Fading."

  APPROACH: Nakagami's derivation was semi-empirical. He started from
  extensive measurements of short-wave fading in Japan (HF radio links)
  and sought a distribution that:
    (a) Could fit measured fading data better than Rayleigh alone
    (b) Had a tunable "severity" parameter
    (c) Was mathematically tractable

  STEP 1: DEFINE THE FADING FIGURE m
    Nakagami defined the fading severity parameter as:

      m = Omega^2 / E[(P - Omega)^2] = (E[P])^2 / Var(P)

    where Omega = E[P] is the mean received power and Var(P) is its
    variance. This is the inverse of the normalized variance (squared
    coefficient of variation).

      m >= 0.5 (lower bound from the distribution properties)
      m = 1:    Rayleigh fading (exponential power distribution)
      m = 0.5:  One-sided Gaussian (worst-case fading)
      m -> inf: No fading (deterministic, AWGN channel)
      m > 1:    Less severe than Rayleigh (e.g., dominant LOS component)

  STEP 2: MATCH TO GAMMA DISTRIBUTION
    Given the constraint that the power distribution has mean Omega and
    the fading figure m, the distribution that satisfies these moment
    constraints (and has maximum entropy subject to them) is the
    Gamma distribution with:

      shape parameter = m
      scale parameter = Omega/m

    The PDF of a Gamma(m, Omega/m) random variable is:

      f(P) = (m/Omega)^m * P^(m-1) / Gamma(m) * exp(-m*P/Omega)

    which is exactly Equation 8.

  STEP 3: PHYSICAL JUSTIFICATION
    The Nakagami-m distribution can also be derived from physical
    principles in certain special cases:

    (a) m = 1 (Rayleigh): When the received signal is the sum of many
        independent, identically distributed scattered components with
        no dominant component, the envelope follows a Rayleigh
        distribution by the Central Limit Theorem. The power (envelope
        squared) then follows an exponential distribution, which is
        Gamma(1, Omega) — i.e., Nakagami with m=1.

    (b) m = integer: When the received power is the sum of m independent
        Rayleigh-fading signals (diversity combining), the total power
        follows a Gamma(m, Omega/m) distribution. This is exactly the
        Nakagami-m distribution.

    (c) m = (K+1)^2 / (2K+1): The Nakagami-m distribution closely
        approximates the Rician distribution (which has a dominant LOS
        component with Rician K-factor) when m is related to K by this
        formula. This makes Nakagami a convenient proxy for Rician
        fading without the mathematical complexity of the Rice PDF
        (which involves a modified Bessel function).

  WHY m = 1 IS USED IN THIS SIMULATION:
    Setting m = 1 reduces Nakagami to Rayleigh fading, which assumes:
    - No dominant LOS component in the small-scale fading
    - Many scattered multipath components of similar amplitude
    - Rich scattering environment (urban setting)

    This is appropriate for:
    - NLOS channels: No direct path, so Rayleigh is the standard model
    - LOS channels: Rayleigh is conservative (reality may have a
      dominant component making it Rician). Using m=1 for LOS gives a
      worst-case fading model. A more accurate LOS model would use
      m > 1 (or equivalently, a Rician model with K > 0).

    The choice of m=1 for both LOS and NLOS is a simplification.
    ITU-R M.2135-1 actually specifies different small-scale fading
    characteristics per cluster per scenario, but the openCV2X
    simulator uses a single Nakagami parameter for tractability.

ASSUMPTIONS:
  1. Wide-Sense Stationary Uncorrelated Scattering (WSSUS): The channel
     statistics don't change over the observation window, and scatterers
     at different delays are uncorrelated.

  2. The fading is "rapid" (small-scale): Power variations occur over
     distances on the order of a wavelength (~5 cm at 5.9 GHz). This
     is distinct from large-scale shadowing (Eq. not shown, but
     modeled separately as log-normal).

  3. The instantaneous power samples are independent (or at least
     uncorrelated) between successive fading realizations.

  4. The parameter m is constant across the simulation area (no spatial
     variation of fading severity). In reality, m may vary depending on
     local geometry.

SOURCE:
  Nakagami, M. (1960). "The m-Distribution — A General Formula of
  Intensity Distribution of Rapid Fading." In: W.C. Hoffman (ed.),
  Statistical Methods in Radio Wave Propagation, Pergamon Press,
  pp. 3-36.
