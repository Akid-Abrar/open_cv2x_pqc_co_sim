--------------------------------------------------------------------------------
                        VALUES DEFINED vs CALCULATED
--------------------------------------------------------------------------------

DEFINED IN SUMO FILES (intersection_LOS_X.rou.xml):

  ✓ Density (veh/km) - Explicitly stated in file comments
  ✓ Flow Rate per approach (veh/h) - Explicitly stated in file comments
  ✓ Maximum Speed (km/h) - Defined as maxSpeed="20.12" m/s (= 72.4 km/h)
  ✓ Individual flow splits - Calculated from 80/10/10 ratio:
      - Through movement: 80% of total flow
      - Left turn:        10% of total flow
      - Right turn:       10% of total flow

CALCULATED (derived from defined values):

  • None - All primary parameters are explicitly defined in SUMO configuration


RELATIONSHIP:

  Flow Rate (veh/h) = Density (veh/km) × Average Speed (km/h)

  Verification example (Scenario C):
    2,172 veh/h = 30 veh/km × 72.4 km/h ✓

--------------------------------------------------------------------------------
                         SIMULATION CONFIGURATION
--------------------------------------------------------------------------------

Intersection Geometry:
  - Four-leg intersection (North, South, East, West approaches)
  - Each approach has 3 movement types: through, left turn, right turn
  - All approaches configured identically

Vehicle Type:
  - Type ID:       "car"
  - Acceleration:  2.6 m/s²
  - Deceleration:  4.5 m/s²
  - Length:        5 m
  - Max Speed:     20.12 m/s (72.43 km/h)

Simulation Duration:
  - Begin:         0 seconds
  - End:           400 seconds (configurable)

Turn Movement Distribution:
  - Through (straight):  80%
  - Left turn:           10%
  - Right turn:          10%

--------------------------------------------------------------------------------
                            ANALYSIS NOTES
--------------------------------------------------------------------------------

Current Focus:
  Comparing Scenario 5 (ECDSA_C_LOS) vs Scenario 17 (Falcon_C_LOS)
  - Traffic density: 30 veh/km
  - Channel model: Line-of-sight (LOS)
  - Analysis period: 0-200 seconds

Key Finding:
  Falcon-512 shows 41% relative PDR degradation at Service Level C
  (18.48% PDR vs 31.32% for ECDSA) due to 470% packet size overhead.

Future Work:
  - Lower densities (A, B): Expected to show better Falcon performance
  - Higher densities (D, E, F): Expected to show worse Falcon performance
  - NLOS scenarios: Additional propagation challenges
